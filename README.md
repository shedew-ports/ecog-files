# ECOG Files
## The official Github Repository used for all the game assets in ECOG: Infinite

</div>

> (Sadly, ECOG: Infinite is now discontinued)
---
This repository includes all assets for each game in ECOG: Infinite. Varying from 99 Nights In The Forest to Telekinesis
SOME of these games are ported by me, however, most of them were ported by big UBG creators like @[bubbls](https://github.com/bubbls), @[genizy](https://github.com/genizy), @[a456pur](https://github.com/a456pur), @[3kh0](https://github.com/3kh0), and a lot more.

Due to ECOG: Infinite being discontinued, we have moved onto a new project: "ECOG". Check it out on my profile!
